# 🎙️ AI Voice Appointment Agent with n8n

## 📌 Overview

This project is an AI-powered Voice Appointment Agent developed using **n8n** automation workflows. It is designed to handle appointment scheduling, manage user conversations, and automate booking processes with minimal human intervention.

The workflow is provided as an **n8n JSON file**, allowing users to easily import and customize it within their own n8n instance.

## 🚀 Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open your **n8n** instance.

3. Click **Import Workflow**.

4. Select the provided `workflow.json` file.

5. Configure your required credentials and API keys.

6. Activate the workflow and start using the Voice Appointment Agent.
